# EdificeREST
REST API to provide functionality for Edifice

## Running locally
1. Install node.js v5.11.0 or later
1. Run `npm install`
1. Run `npm start`

## Running in production
1. `NODE_ENV=production NODE_CONFIG_DIR=./EdificeREST/config pm2 start EdificeREST/app.js`
